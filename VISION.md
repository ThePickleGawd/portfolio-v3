# Dylan Lu — Personal Website Vision

## Aesthetic Direction

**Anime cel-shaded energy** — inspired by JJK Hollow Purple and Demon Slayer's iconic flame effects. Dark background with vibrant, saturated colors rendered in a painterly cel-shaded style with hard edges, color banding, and brush-stroke textures.

### Color Palette
- **Blue orb**: `#2288ff` (vivid blue) with `#44aaff` particles
- **Pink orb**: `#ff2266` (vivid crimson-pink) with `#ff5588` particles
- **Merge purple**: `#8833ee` (deep violet)
- **Explosion**: `#aa55ff` (bright purple)
- **Swirl paints**: `#3399ff` (vivid blue), `#ff3377` (vivid pink), `#9944ff` (vivid purple)
- **Background**: Pure black `#000000`

### Visual Quality (Demon Slayer Style)
- **Cel-shaded orbs**: Distinct color bands (4-step quantized lighting) instead of smooth gradients
- **Hard edge outlines**: Sharp Fresnel rim with anime-style colored outline
- **Flame aura**: Wispy, animated flame tendrils (vertex-displaced with noise) replacing soft gaussian glow
- **Brush-stroke texture**: Quantized noise breaks up smooth surfaces for a hand-painted look
- **Hard specular**: Sharp white-hot anime flash highlights (step function cutoff)
- **Ember particles**: Hard-edge sparks instead of soft fuzzy dots
- **Domain-warped swirl**: Cel-shaded color flows for the merge phase
